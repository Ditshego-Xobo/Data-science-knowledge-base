\# SQL Basics for Data Science



SQL (Structured Query Language) is used to manage and query data stored in databases.



\## Why SQL is Important

\- Allows you to retrieve data from databases  

\- Helps filter and organize data  

\- Used together with Python in Data Science  



\## Basic SQL Commands



\- SELECT – used to retrieve data  

\- FROM – specifies the table  

\- WHERE – filters data  



\## Example



```sql

SELECT name, age

FROM customers

WHERE age > 25;
